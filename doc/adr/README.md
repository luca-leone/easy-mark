# Architecture Decision Records

ADRs use a compact MADR-compatible structure and filenames of the form `NNNN-kebab-case-title.md`.

Required sections:

- `# ADR-NNNN: Title`
- `## Status`
- `## Context`
- `## Decision`
- `## Consequences`
- `## Alternatives Considered`

Accepted records are historical. Do not rewrite a decision to describe a new architecture. Add a new ADR, mark the old record `Superseded by ADR-NNNN`, and link both records.
