# Introduzione

Questa applicazione trasforma i documenti Markdown in frammenti HTML conservati esclusivamente in memoria.

## Navigazione

Usa il menu per aprire i documenti senza ricaricare l'app shell.

## Aggiornamenti live

Modificando un file Markdown, il contenuto e il menu vengono rigenerati e il browser si aggiorna.

Vai alla [guida](guide/getting-started.md).
